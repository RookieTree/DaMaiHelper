## 操作步骤
1. 先去演唱会主页预选好场次、价格还有观演人，点击想看
2. （可选）手机后台杀掉大麦app任务
3. 打开辅助app，给于对应权限
4. （可选）输入歌手名字，默认五月天
5. 点击开抢按钮即可
6. 如果点击开抢后，页面未开始自动跳转，可手动杀死大麦，再次切到辅助app点击开抢<br>
如果想终止辅助app，点击右上角悬浮窗<点击停止>即可
